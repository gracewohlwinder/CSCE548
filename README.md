# Setup

1. Clone the git repo
2. Install MySQL and run database_init.sql.
3. Set up a ‘csce548’ user with the password specified in Database.py, and grant the user
access to the new database.
4. Install Python with the mysql-connector-python and fastapi packages.
5. Install node.js
6. In a new command window, run ‘python -m fastapi dev API.py’ to start the API
7. In a new command window, cd into the ‘frontend’ directory and run ‘npm run dev’ to
start the front end
8. Navigate to the local website at the url and port listed by npm vite.
